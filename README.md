# 贪吃蛇小游戏 · Snake Mini-Game

A classic Snake game built with HTML5 Canvas and vanilla JavaScript — no dependencies, runs directly in any browser.

## 🎮 How to Play

1. Open `index.html` in your browser (or serve with any static file server).
2. Click **开始游戏 START** or press **Space** to start.
3. Control the snake:
   - **↑ W** — up
   - **↓ S** — down
   - **← A** — left
   - **→ D** — right
   - Arrow keys also work
4. Eat the red food 🔴 to grow longer and earn points.
5. Avoid hitting the walls or your own tail!

## ✨ Features

- Score tracking with local-storage **high score**
- Smooth glow/shadow effects on the canvas
- Gradient-coloured snake body
- Start screen & game-over overlay
- Bilingual UI (中文 / English)
